BreedPicks

Description: An app to learn about different dog breeds and save favorite ones. 

Features: allows you to see various facts about hundreds of dogs breeds and add them to a personally named list. 

Obstacles: formatting the DetailView and decoding the data. 

Future additions: adding the option to create multiple personally named lists (for different user needs)
